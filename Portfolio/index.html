My Project

I have done a mini project on “Supermarket Management System” during my BSc where C++ is used as working and interface and file structures as storage.

Using Computer Software

To get rid of manual entry and to replace old paper work system

Managing and Maintaining Supermarket

To update prices and quantity of the products available,for billing purposes...

I have developed "Supermarket Management System" to get rid from manual entry, and instead use computer software to store it in a file. Here we perform various operations like inserting a record, deleting a record, modifying a record, search and displaying a record. For this purpose, need a software which can easily and conveniently maintain essential store details.

The software needs 3 files each for different types of data set. This project "Supermarket Management System" includes some facilities such as easy billing for the billing staff, easy updating of the inventory for the inventory staff, and easy overall managing and updating prices for the Manager.
